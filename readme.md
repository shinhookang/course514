## Course Description

This course introduces the foundations of Artificial Intelligence, covering classical machine learning, deep learning, and modern AI applications. Students will gain both theoretical understanding and practical skills through hands-on coding assignments using Python.

**Prerequisites:** Python programming, Linear Algebra, Statistics (basic)

---

## Schedule

| Week | Topic | Contents | Lab |
|---|---|---|---|
| 1 | Environment Setup | Python install, virtual environment, Jupyter | Python & venv setup |
| 2 | Introduction to AI | History of AI, types of AI, current landscape | — |
| 3 | Python for AI | NumPy, Pandas, Matplotlib essentials | Data exploration |
| 4 | ML Fundamentals | Supervised / unsupervised / reinforcement learning, data pipeline | PyTorch Dataset & DataLoader |
| 5 | Linear Models | Linear regression, gradient descent, loss functions | House price prediction (PyTorch) |
| 6 | Classification | Logistic regression, decision boundary, evaluation metrics | Iris / MNIST (PyTorch) |
| 7 | Tree-based Models | Decision tree, random forest, ensemble methods | Tic-Tac-Toe endgame classification (PyTorch) |
| 8 | **Midterm** | Weeks 1–7 review & exam | — |
| 9 | Hyperparameter Optimization | Search strategies (grid, random, Bayesian), Optuna framework, pruning | Optuna study on MLP |
| 10 | Neural Networks | Perceptron, MLP, backpropagation, activation functions | PyTorch intro |
| 11 | Training Deep Networks | Optimizers (SGD, Adam), batch norm, dropout, regularization | MNIST with MLP |
| 12 | Convolutional Neural Networks | CNN architecture, pooling, feature maps, image classification | CIFAR-10 |
| 13 | Recurrent Neural Networks | RNN, LSTM, GRU, sequence modeling | Time-series / text |
| 14 | Attention & Transformer | Self-attention, multi-head attention, Transformer architecture | Sentiment analysis |
| 15 | Transfer Learning & Modern AI | Pre-trained models, fine-tuning, LLMs, generative AI, AI ethics | Image classifier (ResNet) + API |
| 16 | **Final Project** | Student project presentations | Project demo |
