* **Instructor:** Shinhoo Kang

* **Office Hours:** On-demand (email)

## Course Description

This course introduces the foundations of Artificial Intelligence, covering classical machine learning, deep learning, and modern AI applications. Students will gain both theoretical understanding and practical skills through hands-on coding assignments using Python.

**Prerequisites:** Python programming, Linear Algebra, Statistics (basic)

---

## Schedule

| Week | Topic | Contents | Lab |
|---|---|---|---|
| 0 | Environment Setup | Python install, virtual environment, Jupyter | Python & venv setup |
| 1 | Introduction to AI | History of AI, types of AI, current landscape | — |
| 2 | Python for AI | NumPy, Pandas, Matplotlib essentials | Data exploration |
| 3 | Neural Networks | Perceptron, MLP, backpropagation, activation functions | PyTorch intro |
| 4 | Training Deep Networks | Optimizers (SGD, Adam), batch norm, dropout, regularization | MNIST with MLP |
| 5 | ML Fundamentals | Supervised / unsupervised / reinforcement learning, data pipeline | PyTorch Dataset & DataLoader |
| 6 | Linear Models | Linear regression, gradient descent, loss functions | House price prediction (PyTorch) |
| 7 | Classification | Logistic regression, decision boundary, evaluation metrics | Iris / MNIST (PyTorch) |
| 8 | **Midterm** | Weeks 1–7 review & exam | — |
| 9 | Tree-based Models | Decision tree, random forest, ensemble methods | Tic-Tac-Toe endgame classification (PyTorch) |
| 10 | Hyperparameter Optimization | Search strategies (grid, random, Bayesian), Optuna framework, pruning | Optuna study on MLP |
| 11 | Convolutional Neural Networks | CNN architecture, pooling, feature maps, image classification | CIFAR-10 |
| 12 | Recurrent Neural Networks | RNN, LSTM, GRU, sequence modeling | Time-series / text |
| 13 | Attention & Transformer | Self-attention, multi-head attention, Transformer architecture | Sentiment analysis |
| 14 | Transfer Learning | Pre-trained models, fine-tuning, feature extraction | Image classifier (ResNet) |
| 15 | Modern AI Applications | LLMs, generative AI, AI ethics & responsible use | API / prompt engineering |
| 16 | **Final Project** | Student project presentations | Project demo |
